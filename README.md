# 🚗 EcoDrive - Sustainable Game Jam 2026

**EcoDrive** is a premium, web-based idle/action game where your mission is to transform a polluting, old vehicle into a state-of-the-art 100% electric car. Help clean the world by collecting coins and completing ecological challenges!

![EcoDrive Preview](https://via.placeholder.com/800x400/1e293b/10b981?text=EcoDrive+Sustainable+World)

## ✨ Features

- **Dynamic Evolutive World**: Watch the environment change as you upgrade! The more sustainable your car, the more trees, houses, and cleaner skies appear on the road.
- **Sustainability Upgrade System**:
  - **Exhaust Filter**: Reduces smoke and speeds up coin spawn.
  - **Eco-Friendly Tires**: Doubles coin value and improves car appearance.
  - **Sustainable Materials**: Further doubles coin value and enhances the interior.
  - **Hybrid Engine**: Drastically reduces pollution and moves faster.
  - **100% Electric Battery**: The peak of sustainability! Clears the world and unlocks the ending.
- **4 Interactive Minigames**:
  - **Waste Sorting**: Sort recyclable items (Paper, Plastic, Metal, Glass, Organic) into the correct bins.
  - **Tree Planting**: Dig, plant, and water your own tree to increase sustainability.
  - **Eco-Knowledge Quiz**: Test your knowledge with various randomized questions.
  - **Sustainable Choice**: Make the right real-world choices (Bicycle vs Car, LED vs Incandescent, etc.).
- **Global Localization**: Full support for **Portuguese** and **English**, switchable at any time.
- **Developer Mode**: A hidden mode for testing with infinite coins and free upgrades.
- **Premium Design**: Modern Glassmorphism UI, smooth Parallax background, and micro-animations for an immersive experience.

## 🛠️ Technologies Used

- **HTML5**: Semantic structure.
- **CSS3**: Premium styling, custom animations, and responsive layout.
- **Vanilla JavaScript**: Core logic, game state machine, and Canvas drawing API.
- **LocalStorage**: Your progress is automatically saved!

## 🎮 How to Play

1. **Collect Coins**: Click on the coins appearing on the road.
2. **Visit the Workshop**: Upgrade your car whenever you have enough coins.
3. **Handle Alerts**: When the "ALERT!" button appears, click it to start a minigame and earn massive bonuses.
4. **Answer Correctly**: Minigame bonuses are reduced if you make mistakes—stay sharp!
5. **Electric Future**: Reach the 100% Electric Battery to win the game.

## 🚀 Running Locally

1. Clone or download this repository.
2. Open `index.html` in any modern web browser.
3. No build step or installation required!
